# 更新日志

## [1.0.0] - 2026-04-03

### 首次发布

- **多供应商支持**
  - 支持 Infini AI、MiniMax 等 Coding Plan 供应商
  - 供应商切换功能

- **配置共享**
  - 与 CLI 工具共享 `~/.codingplan-config.json` 配置文件
  - 统一凭据管理，CLI 和扩展使用相同配置

- **实时状态监控**
  - 状态栏显示供应商名称、用量百分比、剩余时间
  - 智能颜色提示（绿/黄/红）

- **双语支持**
  - 中文/英文界面切换

- **侧边栏集成**
  - 快速访问设置和帮助
  - Token 消耗统计显示

# 更新日志

## [1.2.9] - 2026-03-24

### 优化

- Tooltip 显示全面改版
  - 采用 Markdown list 格式，兼容性更好
  - 模型名称简化显示（Hailuo-2.3-Fast → Hailuo，speech-01 → speech-hd）
  - MiniMax-M* 模型显示周限额状态
- 状态栏周配额显示优化
  - 无限制套餐显示 ♾️ 符号，不再显示 0%
- Token 消耗统计移至 tooltip 显示
  - 显示昨日消耗、近7天消耗、套餐总消耗
  - tooltip 底部正确换行，避免与统计信息重叠

## [1.2.5] - 2026-03-18

### 新增功能

- 新增周维度用量支持
  - 状态栏新增周用量显示（中文：`周`，英文：`W`）
  - 示例：`⏱ 25m 4% · 周 1%` / `⏱ 25m 4% · W 1%`

### 优化

- 英文版 tooltip 数值单位优化（K/M/B）
- 中文版 tooltip 数值单位保留（万/亿）
- 提升数值显示精度至两位小数

### Bug 修复

- 修复 Weekly Reset 文本未翻译的问题（中文：`4 天 4 小时后重置`，英文：`4d 4h until reset`）
- 修复加载状态文字未翻译的问题（中文：`加载中...`，英文：`Loading...`）

---

## [1.2.4] - 2026-03-16

### Bug 修复

- 修复套餐总消耗计算错误的问题
  - 发现 `current_credit_reload_time` 实际为下次充值日期（与到期时间相同），而非当前周期开始时间
  - 改为从套餐到期时间精确回推 1 个月计算周期开始时间
  - 使用套餐到期时间作为计算终止点，而非当前时间
  - 提升账单获取上限至 5000 条（原 1000 条），覆盖更多历史记录
